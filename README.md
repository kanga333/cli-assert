# cli-assert

Simple command line assert tool

```text
# cli-assert -h
Usage of ./cli-assert:
  -code int
    Expected return code
  -not-stderr value
    String not contained in stderr
  -not-stderr-regex value
    Regex not matching stderr
  -not-stdout value
    String not contained in stdout
  -not-stdout-regex value
    Regex not matching stdout
  -stderr value
    String contained in stderr
  -stderr-regex value
    Regex matching stderr
  -stdout value
    String contained in stdout
  -stdout-regex value
    Regex matching stdout
```
